# Arsany Attalla Portfolio

A modern, interactive personal portfolio built with **Next.js**, **React**, **Tailwind CSS**, and **Framer Motion**. Features an animated signature intro, smooth scroll navigation, and responsive design.

---

## Demo

https://arsanyattalla.com
---

## Features

- Animated signature intro for a personal touch
- Smooth scroll navigation across sections: Home, About, Projects, Resume, Contact
- Responsive design optimized for mobile and desktop
- Projects section showcasing work with live links
- Resume section with tabs for Skills, Experience, and Education
- Contact section with social media and email links
- Built using modern frontend tools: Next.js, React, Tailwind CSS, Framer Motion
